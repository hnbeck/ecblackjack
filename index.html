Prolog Black Jack

This simple Black Jack like game is intended as example for projects using SWI Prolog, its Pengines library and Tau-Prolog. Graphics is done with JS Processing variant p5js. This is no Casino game, there is nothing to win and no purpose. It's just a demo - for experimenting with. Soucecode is available at https://github.com/hnbeck/ecblackjack with no warranties.

Click on the deck for drawing a card or click "Stand"-button to give the dealer the control. Do a page refresh for a new game.
